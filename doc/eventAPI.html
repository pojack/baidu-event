    chart api

    获取图表坐标点数组接口

    /event/chart get

    get

    得到图表上的坐标点对象数组

    get /event/chart

    得到图表上的坐标点对象数组

    • Request
    • Response

    Query Parameters

    • source: required (string)

      来源：百度贴吧

      Example:

      百度贴吧
    • data: required (string)

      数据类型：跟帖量 或 发帖量

      Examples:

      example:

      跟帖量

      example1:

      发帖量
    • beginTime: required (string)

      开始日期：格式 日/月/年

      Example:

      05/01/2017
    • endTime: required (string)

      结束日期：格式为 日/月/年

      Example:

      05/01/2017
    • table: required (string)

      数据来源表：specialEvent 或dailyEvent或handledEvent

      Examples:

      example:

      specialEvent

      example1:

      dailyEvent

      example2:

      handledEvent

    HTTP status code 200

    获取图表成功

    Body

    Media type: application/json

    Type: array of items

    Items: items

    • x: required (string)

      横坐标

    • y: required (integer)

      纵坐标

    Example:

    [{"x":"2017-05-01","y":6},{"x":"2017-05-02","y":15},{"x":"2017-05-03","y":0}]
